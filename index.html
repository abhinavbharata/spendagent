iOS Shortcut setup
A one-tap Share Sheet shortcut that captures any bank notification and appends it to a file — ready to upload here anytime.
Step by step
1
Open the Shortcuts app on your iPhone and tap + to create a new shortcut.
2
Tap Add action → search Receive input from Share Sheet → select it. Set input type to Text.
3
Tap + again → search Append to File → select it. Set file name to bank-notifications.txt, location to iCloud Drive.
4
Enable Make New Line so each notification is on its own line.
5
Tap the shortcut name at top and rename it Save bank notification. Tap Done.
6
Test: open any bank SMS → long-press → Share → tap Save bank notification. Check iCloud Drive for the file.
7
Weekly: upload bank-notifications.txt from iCloud Drive via the Add tab above.
What you can capture
SMS alerts — long-press message → Share → Save bank notification
Push notifications — swipe down banner → share icon → Save bank notification
Email alerts — open email → share icon → Save bank notification
